    Installation (Debian based distributions)

    $ sudo wget http://cytral.github.io/twp/tools/install.sh -O - | bash

    Get Started

    1. Configure the app, see the file 'twp.conf'
    2. Run or restart the app
    3. Login Admin using default credentials:
      • User: admin
      • Password: admin
    4. Go to settings page and change admin password
