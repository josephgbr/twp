+ Teeworlds Web Panel
=============================================
TWP is an app-web for manage your local servers. Inspired by the work of 'LXC Web Panel'.

The project uses:
- SQLite v3.x
- Flask v0.10.1
- Requests v2.8.1
- Teeworlds Library (by m!nus)
- Bootstrap v3.3.5
- JQuery v1.11.3
- Font Awesome v4.4.0
